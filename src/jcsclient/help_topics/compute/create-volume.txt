
NAME :
    create-volume


DESCRIPTION :
    Creates Volume in your JCS account. An empty volume can be created using 
    size paramater. Volume can be created from an existing snapshot as well.
    Size can also be passed as a paramater while creating volume from snapshot.


SYNOPSIS :
    create-volume
    [--size <integer>]
    [--snapshot-id <value>]


OPTIONS :
    [--size <integer>]
    Size of the volume that needs to be created

    [--snapshot-id <value>]
    Id of Snapshot using which the volume needs to be created

    [--encrypted <boolean>]
    Specify this to allow volume to be encrypted

    [--volume-type <value>]
    Specify standard for hdd volume, ms1 for ssd volumes. Default is standard.

EXAMPLES :
    jcs compute create-volume --size 5

    jcs compute create-volume --size 10 --encrypted true

    jcs compute create-volume --snapshot-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf
    
    jcs compute create-volume --snapshot-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf --size 100

    jcs compute create-volume --size 20 --volume-type ms1

    jcs compute create-volume --size 10 --encrypted true --volume-type standard

    jcs compute create-volume --size 5 --encrypted true --volume-type ms1

    jcs compute create-volume --snapshot-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf --volume-type ms1
